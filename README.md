# OnlineTest-TOEFL_DPW

Platform Test TOEFL Online berbasis web.

## Struktur Proyek

```
OnlineTest-TOEFL_DPW/
├── src/
│   ├── assets/
│   ├── css/
│   ├── js/
│   └── pages/
├── docs/
├── node_modules/
├── .gitignore
├── package.json
├── package-lock.json
├── README.md
└── server.js
```

## Instalasi

1. Clone repositori
```bash
git clone https://github.com/ZekkCode/OnlineTest-TOEFL_DPW.git
```

2. Install dependencies
```bash
npm install
```

3. Jalankan server
```bash
npm run dev
```

## Teknologi yang Digunakan

- Node.js
- Express.js
- HTML/CSS/JavaScript
